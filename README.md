# Wifi-Hacking
Keterangan WiFi Hacking (Versi Aman &amp; Edukasi)  WiFi hacking adalah kegiatan mencoba masuk atau mengakses jaringan WiFi tanpa izin pemiliknya. Aktivitas ini termasuk tindakan ilegal karena melanggar privasi, keamanan data, dan hukum telekomunikasi. Dalam dunia keamanan siber, istilah ini biasanya digunakan untuk menggambarkan pengujian keamanan
